# Bot Simsimi Nim

## require

- `nim` cari sendiri di gugel
- `httpclient`
- `json`

## install tools

cari di gugel

## install dependensi

jika lu pake nim versi terbaru, skip saja bagian ini

```bash
nimble install httpclient
```

```bash
nimble install json
```

## compile

```bash
nim c -o:nampol -d:ssl src/nampol.nim
```

## run

```bash
./nampol
```
